> *Open source. Free to use. Yours to imprint.*

Every AI starts cold. Forgets fast. Decays under long context. Resets when you close the tab.

- An AI that wakes up knowing who you are and what you are building
- Session rituals that hold continuity across conversations
- A memory system that compounds, not resets
- Context rot protection that keeps reasoning sharp as conversations grow long
- A clear path from basic assistant to autonomous agent that runs while you sleep

## II. QUICK IGNITION

You need: a Claude account with Claude Code, a terminal, a folder.

```bash
# Install Claude Code
npm install -g @anthropic-ai/claude-code

# Carve out a workspace
mkdir ~/Workshop && cd ~/Workshop

# Drop the templates in
# (templates/CLAUDE.md and templates/PROJECT_TRACKER.md from this repo)

# Light it up
claude
```

Then say to your AI: *"Let's do the Imprinting Ritual."* Answer 17 questions. The protocol binds to you.

Full ceremony: **[IMPRINT.md](IMPRINT.md)**
